What is Descriptive Statistics

Descriptive statistics describe, show, and summarize the basic features of a dataset found in a given study, presented in a summary that describes the data sample and its measurements. It helps analysts to understand the data better.


Descriptive statistics represent the available data sample and does not include theories, inferences, probabilities, or conclusions. Thatâ€™s a job for inferential statistics.


Types of Descriptive Statistics

Distribution, (which means with each value's frequency)

Central Tendency, (which covers the averages of the values)

Variablity or Dispersion, (which shows how spread out the values are)

Population and Sample

Population refers to the entire group of individuals about whom you wish to draw conclusions. The sample refers to the group of people from which you will be collecting data.

Population Data

In statistics, population is the entire set of items from which you draw data for a statistical study. It can be a group of individuals, a set of items, etc. It makes up the data pool for a study.
